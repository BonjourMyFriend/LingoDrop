# LingoDrop

桌面沉浸式翻译学习助手 | Desktop Immersive Learning Assistant

---

## 简介

LingoDrop 是一款专为语言学习者设计的桌面工具，支持**翻译学习**与**商务英语改写**两大核心功能。

### 翻译学习

- 全局热键（默认 `Ctrl+Space`）随时呼起
- 自动检测输入语言，智能翻译为目标语言
- 翻译结果附带词汇解析、语法拆解、IPA 音标
- 支持 Markdown 实时渲染
- TTS 语音朗读，边看边听

### 商务改写

- 一键将口语化英文转换为专业商务英语
- 自动复制并粘贴替换原文

### 主要特性

- 🎯 **智能语言检测**：自动识别母语/目标语/第三方语言
- 📝 **专业词汇解析**：包含词源标注、音标、词性、常见搭配
- 🔧 **高度可定制**：支持自定义系统提示词、行业领域上下文
- 🌐 **多语言支持**：中文、英文、韩语、日语、德语、法语、西班牙语等
- 🎨 **Notion 风格 UI**：简洁优雅的深色主题
- ⌨️ **全局热键**：任意窗口下均可使用

---

## 技术栈

- Python 3
- PyQt6（GUI 与系统托盘）
- OpenAI API（LLM 集成）
- edge-tts（微软 TTS 语音合成）
